# Fooll-cli
Command line tools for [Fooll framework](https://github.com/fooll/fooll)

## Installation
```
sudo npm i -g fooll-cli
```

## API

### Generate application boilerplate
```
fooll -a appName
```
**Or**, the long version:
```
fooll --create_app appName
```

### Create application module
```
fooll -M moduleName
```
**Or**, the long version:
```
fooll --app_module moduleName
```
